# Changelog

All notable changes to the "vscode-git-blame-info" extension will be documented in this file.

## [1.0.0] - 2026-04-01

### Added

- Color-coded git blame annotations in the editor gutter, inspired by Eclipse EGit
- Right-click line number gutter to show/hide revision information
- Hover popup with full commit details (author, date, commit ID, summary)
- Open Commit action to view commit diff in the editor
- Open History action to browse the file's full commit history with diff navigation
- Compare with Current File from the history view
- Copy Commit ID to clipboard via context menu
- Settings button in hover popup for quick access to extension configuration
- Configurable annotation display: author, date, commit ID, summary
- Relative date display option (e.g. "2 months ago")
- Customizable date format and column width
- Dynamic color assignment with theme-aware light/dark support
